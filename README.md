### 🛡️ Sobre el Sistema
**GiyuTomioka-Bot** es una herramienta de alto rendimiento diseñada para WhatsApp. Bajo la disciplina del Pilar del Agua, este bot mantiene la paz en tus grupos eliminando amenazas automáticamente las 24 horas. Hecho con honor en **Venezuela 🇻🇪**.

### 🌊 Las Posturas (Funciones Principales)
* 💧 **Primera Postura: Anti-Links** - Expulsión inmediata de enlaces no autorizados.
* 💧 **Segunda Postura: Anti-Spam** - Control de flujo de mensajes para evitar retrasos.
* 💧 **Tercera Postura: Economía** - Sistema de minería, pesca y niveles (`daily`, `minar`).
* 💧 **Cuarta Postura: Creatividad** - Generación de stickers y herramientas multimedia.
* 💧 **Quinta Postura: Inteligencia** - Respuestas avanzadas mediante IA (ChatGPT).

### 🚀 Instalación Rápida
Este proyecto está configurado para un despliegue limpio en Railway o Termux, optimizando el uso de memoria.

```bash
git clone [https://github.com/GabrielVB26/GiyuTomiokaBot-MD](https://github.com/GabrielVB26/GiyuTomiokaBot-MD)
cd GiyuTomiokaBot-MD
npm install
npm start
